# OpenClaw — Sri's Setup

This is my working OpenClaw instance, built on top of the [OpenClaw Mastery for Everyone](course/README.md) course by Aishwarya Reganti & Kiriti Badam.

The course covers Days 1-10. Everything beyond that is my own extension — adding RAG, self-improvement, automated research ingestion, and scheduled opportunity scouting.

## What's Running

- OpenClaw on a Hetzner VPS (24/7)
- Telegram as the primary channel
- Gmail triage with injection protection
- Web research via agent-browser
- ChromaDB-backed RAG pipeline for paper ingestion and retrieval
- Cron jobs: daily reflection, learning reports, AI opportunities scout
